
# 🌿 מתכנן הגינה שלי - My Garden Planner

אפליקציה אינטראקטיבית לתכנון גינות המיועדת לתלמידי כיתות ד'-ה', המשלבת בינה מלאכותית (Google Gemini) לחישוב חומרים והצעות עיצוב.

## ✨ תכונות עיקריות
- **תכנון מדויק**: גרירה והנחה של ערוגות, עצים, גדרות ושולחנות.
- **קנה מידה ידידותי**: מותאם לילדים - כל 5 משבצות שוות למטר אחד (משבצת = 20 ס"מ).
- **דוח חומרים חכם**: חישוב אוטומטי של ליטרים אדמה, טוף וכמות בלוקים נדרשת.
- **בינה מלאכותית**: קבלת הצעות לתכנון הגינה בלחיצת כפתור.
- **שמירה מקומית**: התוכנית נשמרת אוטומטית בדפדפן.
- **ייצוא לתמונה**: אפשרות לשמור את התכנון כקובץ SVG להדפסה.

## 🛠 טכנולוגיות
- **React 19**
- **Tailwind CSS** (עיצוב רספונסיבי)
- **Google Gemini API** (ייעוץ חכם)
- **LocalStorage API** (שמירת נתונים)

## 🚀 איך להפעיל?
הפרויקט בנוי כ-ES Module מודרני. ניתן להריץ אותו ישירות מכל שרת סטטי (כמו GitHub Pages).

1. העלו את הקבצים ל-GitHub.
2. הגדירו ב-Settings של ה-Repo את GitHub Pages שיצביע על ה-Branch הראשי.
3. האפליקציה תהיה זמינה בכתובת ה-URL שלכם!
